Как добавить новую презентацию
01
Даёшь задачу Perplexity
Ссылка на этот репо + описание новой презентации (тема, ивент, ключевые тезисы)
02
Perplexity читает DESIGN.md
Агент забирает дизайн-гайд, создаёт HTML-презентацию в стиле VTech через Reveal.js
03
Pull Request в GitHub
Perplexity создаёт PR с новой папкой presentations/ и meta.json
04
Merge → Автодеплой
После принятия PR — GitHub Actions деплоит сайт на GitHub Pages автоматически
