Как добавить презентацию

01
Напишите Perplexity
Дайте ссылку на репозиторий и опишите новую презентацию в одном сообщении.
Репо: github.com/olegbuselru/vtech-presentations Тема: «DevRel метрики 2026» Дата: 2026-05-10 Категория: devrel
02
Perplexity создаёт PR
Ассистент генерирует файл презентации по DESIGN.md и открывает Pull Request в репозиторий.
presentations/devrel-metrics-2026/ ├── index.html └── meta.json
03
Вы мёрджите PR
Просмотрите и примите Pull Request на GitHub. GitHub Pages обновит сайт автоматически.
✓ PR merged → GitHub Actions ✓ Site deployed in ~2 min
04
Карточка на сайте
Новая презентация появляется в галерее автоматически — карточка, дата, категория, ссылка.
// presentations/index.json обновляется // галерея читает его при загрузке
